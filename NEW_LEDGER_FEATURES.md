# 圆聚助残平台 - 新增账本子应用功能

## 🎯 功能概述

为了更好地服务农村残疾人和残疾人群体，我们在原有的11类记名账本基础上，新增了3个专项账本子应用：

1. **病历账本** - 记录医疗历史，守护健康人生
2. **梦境账本** - 记录美好梦想，照亮人生方向  
3. **实账本** - 记录真实经历，见证人生历程

## 🏗️ 技术架构

### 数据库Schema更新

```prisma
model NamedLedger {
  // 新增字段
  specialData String?  // JSON格式存储特殊数据
  tags        String?  // JSON数组存储标签
  privacy     String   @default("private") // public, private, selective
  
  // 扩展账本类型
  // medical, dream, reality (新增三种)
  ledgerType  String
}
```

### API端点

- `GET/POST /api/ledger/medical` - 病历账本API
- `GET/POST /api/ledger/dream` - 梦境账本API  
- `GET/POST /api/ledger/reality` - 实账本API

### 页面路由

- `/ledger/medical` - 病历账本页面
- `/ledger/dream` - 梦境账本页面
- `/ledger/reality` - 实账本页面
- `/ledger` - 记名账本主页（包含子应用入口）

## 📋 功能详解

### 1. 病历账本 (Medical Ledger)

**核心功能：**
- 记录疾病类型、病情起因、诊断结果
- 记录治疗方案、用药记录、就医医院
- 评估严重程度、是否慢性病、是否需要帮助
- 支持隐私设置和状态管理

**特殊数据结构：**
```json
{
  "diseaseType": "髋关节重伤",
  "cause": "10岁时被村霸儿子从房檐推下摔伤",
  "diagnosis": "右腿髋关节粉碎性骨折，导致瘫痪",
  "treatment": "长期康复治疗，物理治疗，药物治疗",
  "severity": "severe",
  "isChronic": true,
  "needsHelp": true,
  "helpType": "需要康复设备费用，护理服务"
}
```

**价值计算：**
- 基础价值：10分
- 严重程度：轻微+0，中等+10，严重+20
- 慢性病：+15分
- 需要帮助：+10分
- 详细记录：+15分

**天枰倾斜机制：**
- 残疾人用户：获得价值30%的共同体账户庇佑
- 体现"健全人向残疾人倾斜"理念

### 2. 梦境账本 (Dream Ledger)

**核心功能：**
- 记录各种类型的梦想（生活、职业、健康、家庭、创新、社会）
- 设定目标实现时间和行动计划
- 记录所需资源和面临的障碍
- 支持分享意愿和合作需求

**特殊数据结构：**
```json
{
  "dreamType": "health",
  "dreamTitle": "重新站立的梦想",
  "description": "通过先进的医疗技术和康复训练，希望能够重新恢复行走能力",
  "targetDate": "2026-12-31",
  "requiredResources": ["先进医疗技术", "康复设备", "专业护理"],
  "currentProgress": 15,
  "actionSteps": ["寻找更好的医疗资源", "进行康复训练"],
  "collaboration": true
}
```

**价值计算：**
- 基础价值：15分
- 梦想类型：创新+25，社会+20，职业+15
- 优先级：高+15，中+10
- 详细规划：+25分
- 合作意愿：+15分

**天枰倾斜机制：**
- 所有用户：获得价值20%的共同体账户庇佑
- 健全人用户如有合作意愿：额外获得10%投资点数
- 体现"城市人向农村人倾斜"理念

### 3. 实账本 (Reality Ledger)

**核心功能：**
- 记录真实生活事件和经历
- 支持多种事件类型（日常、奋斗、成就、改变、真相）
- 记录情感影响和学到的教训
- 支持证据材料和见证人

**特殊数据结构：**
```json
{
  "realityType": "struggle",
  "eventTitle": "22年自学之路",
  "eventDate": "2003-01-01",
  "detailedDescription": "通过互联网自学编程、设计、网络技术等",
  "emotionalImpact": "虽然过程艰难，但获得了知识和技能",
  "lessonsLearned": "身体残疾不能阻挡学习",
  "evidence": ["学习笔记", "作品集", "证书"],
  "authenticity": "verified",
  "impact": "personal"
}
```

**价值计算：**
- 基础价值：12分
- 事件类型：奋斗+20，成就+18，改变+15，真相+25
- 影响范围：社会+20，社区+15，家庭+10
- 详细记录：+32分
- 真实性验证：+15分

**天枰倾斜机制：**
- 残疾人用户：获得价值40%的共同体账户庇佑
- 健全人用户：获得价值15%的投资点数
- 体现"已婚者向未婚者倾斜"理念

## 🔄 工作流程

### 记录创建流程
1. 用户填写表单 → 2. 系统验证数据 → 3. 计算价值 → 4. 创建记录 → 5. 更新账户余额

### 审核验证流程
1. 提交记录（pending）→ 2. 管理员审核 → 3. 验证通过（verified）→ 4. 应用实施（applied）

### 价值体现流程
1. 创建记录 → 2. 获得基础奖励 → 3. 验证后获得额外奖励 → 4. 应用后获得完整价值
